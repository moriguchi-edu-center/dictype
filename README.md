# 辞書とタイピング

児童・生徒向けの**日本語単語帳＋タイピング練習アプリ**です。  
単語と説明を登録し、タイピング練習で定着させながら、忘却曲線に基づく復習提案も行います。

インストール不要。ブラウザで `index.html` を開くだけで使えます。

![アプリアイコン](dictypeimg.png)

## 主な機能

### 単語帳
- **単語・説明・教科・単元**を登録
- データはブラウザの **localStorage** に自動保存
- **辞書ファイル（JSON形式）** の書き出し・読み込み
- **エクセル（.xlsx）** からまとめて単語を追加（テンプレート付き）

### タイピング練習
- 練習モード
  - 単語 → 説明（両方）
  - 単語のみ
  - 説明のみ
  - **思い出し → タイピング**（説明だけ見て単語を思い出す。正解なら次へ、時間切れなら答えを表示してタイピング練習）
- 制限時間を4段階から選択（🐢 かめ / 🐰 うさぎ / 🐆 チーター / ✈️ ジェット）
- 教科・単元で出題を絞り込み
- ランダム出題は**重複なし**（全問終了で終了、自動では繰り返さない）
- 日本語 IME 対応、Unicode 文字単位の判定

### 復習サポート
- **わすれかけている単語**をホーム画面で提案
- 復習間隔：1 → 3 → 7 → 14 → 30 → 60 日
- **忘れないコツ**画面で忘却曲線の説明とグラフを表示

## 使い方

1. このフォルダ内の `index.html` をブラウザで開く
2. 「単語を追加」で単語を登録する（または `words.json` / エクセルを読み込む）
3. 「タイピング」から練習を開始する

### キーボード操作（タイピング中）

| キー | 操作 |
|------|------|
| `Enter` | セットアップ画面で開始 / 結果画面で次の問題 |
| `Tab` | スキップ |
| `Esc` | 終了・メニューに戻る |

## 辞書ファイル（JSON）の形式

```json
{
  "version": 1,
  "words": [
    {
      "id": "rika-001",
      "word": "光合成",
      "description": "植物が光のエネルギーを使って…",
      "subject": "理科",
      "unit": "植物",
      "createdAt": "2026-06-28",
      "lastReviewedAt": "2026-06-28",
      "reviewLevel": 0,
      "nextReviewDate": "2026-06-29",
      "reviews": [{ "date": "2026-06-28", "type": "registered" }]
    }
  ]
}
```

## エクセル一括登録

1. 「単語を追加」→「テンプレートをダウンロード」
2. 1行目に見出し（**単語・説明・教科・単元**）、2行目以降にデータを入力
3. 「エクセルファイルを読み込む」で取り込み

> エクセル機能は [SheetJS](https://sheetjs.com/)（CDN）を利用するため、**インターネット接続**が必要です。

## 動作環境・注意事項

- 推奨：Chrome、Edge、Firefox などのモダンブラウザ
- データは**そのブラウザ内**に保存されます。別の PC やブラウザへ移す場合は JSON 書き出しを利用してください
- `file://` で開いた場合、`words.json` の自動読み込みは制限されることがあります。初回は「辞書ファイル（JSON形式）から読み込み」を使ってください
- 辞書ファイルとの**連携（自動保存）**は File System Access API 対応ブラウザ（主に Chrome / Edge）で利用できます
